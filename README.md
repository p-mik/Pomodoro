# Pomodoro

Osobní Pomodoro tracker s cross-device notifikacemi a statistikami.

**Doména:** [pomodoro.upupaepops.cz](https://pomodoro.upupaepops.cz)  
**Součást:** [upupaepops.cz](https://upupaepops.cz)

## Stack

- Python / Django 5.x
- PostgreSQL 17
- Bootstrap 5.3 + Chart.js
- Docker + Docker Compose
- PWA (installable, Web Push notifikace)
